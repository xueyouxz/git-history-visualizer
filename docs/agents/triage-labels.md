# Triage Labels

工程技能使用五个固定角色。右列是本仓库 GitHub Issues 中的实际标签。

| 工程技能角色 | GitHub 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者判断 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整定义，可由代理实施 |
| `ready-for-human` | `ready-for-human` | 需要人工实施 |
| `wontfix` | `wontfix` | 不处理 |

技能提到角色时，使用本表对应的 GitHub 标签。若仓库以后调整标签词汇，只修改右列。
